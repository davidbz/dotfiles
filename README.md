# Dotfiles and setup

My setup. Using Arch and i3-gaps.

## Getting Started

Read [Arch installation guide] (https://wiki.archlinux.org/index.php/installation_guide)

### Installing

Install the following using pacman:

```
xorg-xinit
xorg
i3-gaps
```
